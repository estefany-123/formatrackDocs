<h2>¿Que son las Sedes?</h2>

Las sedes en el sena son las instalaciones físicas donde se desarrollan los procesos de formación y administrativos de la institución. Cada Centro de Formación puede tener una o varias sedes ubicadas en diferentes lugares para facilitar el acceso a los aprendices.

<h3>Descripcion</h3>

Este endpoint permite listar todas las sedes registradas.

Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/Sede
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a las sedes
