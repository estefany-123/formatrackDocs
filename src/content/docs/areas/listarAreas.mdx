<h2>¿Que son las Areas?</h2>

Las areas son lugares especificos, los cuales cada uno de estos se enfocan en diferentes ambitos como agroecologico que es todo sobre los recursos naturales o el tecnologico que maneja todas las herramientas digitales y sistemas de gestion.


<h3>Descripcion</h3>

Este endpoint permite obtener la lista de áreas disponibles en el sistema.

Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/Areas
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a las areas
