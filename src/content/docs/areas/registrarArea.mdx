<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Content-Type | application/json | Indica que los datos esran manejados en fromato JSON
| Authorization |  Bearer `token` | Token con autorizacion para acceder a las areas

<h3>Endpoint para los Datos del Área</h3>

Para agilizar la creacion de nuevas areas te ofrecemos este endpoint de apoyo para que puedas consultar y obtener el ID de la sede necesaria para crear tu area.
<li><a href="/sedes/listarSedes">Sede</a></li>


<h3>Parametros del cuerpo</h3>

| Parametro y Tipo | Descripcion
|----------|--------
| nombre (string) |  Nombre que se le va a designar a dicha area no mayor a (50)
| persona_encargada (string) | Usuario que dirige el area en general
| estado (boolean) | Si se encuentra activo o inactiva el area
| created_at (date) | Fecha en la que se registro o creo el area
| updated_at (date) | Ultima fecha en la que se le hizo alguna modificacion
| fk_sede (integer) |  ID de la sede en la que estar el area. Para consultar las sedes existentes consulte este endpoint <a href="/sedes/listarSedes">sedes</a>


### Body

```json
{
  "nombre": "string",
  "persona_encargada": "string",
  "estado": "boolean",
  "created_at": "date",
  "updated_at": "date",
  "fk_sede": "integer"
}
```
