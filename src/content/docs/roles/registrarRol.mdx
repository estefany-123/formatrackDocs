<h3>Encabezados de Solicitud</h3>

| Encabezado    | Valor            | Descripción                                        |
| ------------- | ---------------- | -------------------------------------------------- |
| Content-Type  | application/json | Indica que los datos están en formato JSON         |
| Authorization | Bearer `token`   | Token con autorización para acceder a este recurso |

<h3>Parámetros del cuerpo</h3>

| Parámetro y Tipo | Descripción                                     |
| ---------------- | ----------------------------------------------- |
| nombre (string)  | Nombre del rol que se desea registrar           |
| estado (boolean) | Define si el rol se encuentra activo o inactivo |

<h3>Body</h3>

```json
{{
    "nombre": "Aprendiz",
    "estado": true
}}
```
