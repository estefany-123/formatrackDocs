<h2>¿Que son los roles?</h2>

Los roles se refieren a las funciones y responsabilidades que desempeñan las diferentes personas dentro de la institución, ya sea en la formación, administración o gestión de los procesos educativos.

<h3>Descripcion</h3>

Este endpoint permite listar todos los roles registrados.

Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/api/rol/
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los roles
