  <span class="sl-badge success small astro-avdet4wd">GET</span>

</Tabs>

| Encabezado    | Valor            | Descripción                                          |
| ------------- | ---------------- | ---------------------------------------------------- |
| Content-Type  | application/json | Indica que los datos están manejados en formato JSON |
| Authorization | Bearer `token`   | Token con autorización para acceder al recurso       |
