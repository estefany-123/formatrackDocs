Este endpoint permite listar todos los elementos registrados en el inventario.

| Variable | Valor                                      |
| -------- | ------------------------------------------ |
| URL      | [/api/inventario/](/api/inventario/)      |
| Method   | `GET`                                      |
| Headers  | Authorization : Bearer `token`            |
