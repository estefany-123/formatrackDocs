<h2>Â¿Que son los Inventarios?</h2>

Cuando hablamos de Inventarios hacemos referencia a el registro y control de los bienes o elementos formativos que posee cada centro de formacion para el desarrollo de sus programas educativos 

<h3>Descripcion</h3>

Este endpoint permite listar todos los elementos registrados en el inventario.

Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/api/inventario/
  ```
  </TabItem>
</Tabs>


<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los Inventarios
