Este endpoint permite actualizar un elemento del inventario utilizando su ID.

<p> 
<strong>Método: </strong><span class="sl-badge note small astro-avdet4wd">PUT</span>
</p>

<h1>No debe tener imagen</h1>

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/api/inventario/update/:id_inventario
  ```
  </TabItem>
</Tabs>

Es necesario el envío del token de acceso <code>access_token</code> para hacer uso de este endpoint. Para ver cómo generar el token, vea <a href="/usuarios/login">como generar token de acceso.</a></aside>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los inventarios



### Body

```json
{
  "stock": "integer",
  "estado": "string",
  "created_at": "date",
  "updated_at": "date",
  "fk_sitio": "integer",
  "fk_elemento": "integer"
}
```
