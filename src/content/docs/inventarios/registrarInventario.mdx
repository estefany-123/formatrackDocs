Este endpoint permite registrar un nuevo elemento en el inventario con los datos proporcionados.

<p> 
<strong>Método: </strong><span class="sl-badge note small astro-avdet4wd">PUT</span>
</p>

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/api/inventario/register/
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los inventarios


<h3>Parametros del cuerpo</h3>

| Parametro y Tipo | Descripcion
|----------|--------
| stock (integer) | Cantidad de dicho elemento en bodega
| estado (boolean) | Si el elemento esta activo o inactivo
| created_at (date) | Fecha en la que se registro el elemento en el inventario
| updated_at (date) | Fecha en la que se le hizo alguna modificacion en el inventario 
| fk_sitio (integer) | ID del sitio en el que se encuentra el inventario. Para ver los sitios existentes visite el siguiente endpoint <a href="/sitios/listarSitios">Sitios</a>
| fk_elemento (integer) | ID del elemento que seregistrara al inventario. Conozca los elementos disponibles aqui <a href="/elementos/listarElementos">elementos</a>

### Body

```json
{
  "stock": "integer",
  "estado": "string",
  "created_at": "date",
  "updated_at": "date",
  "fk_sitio": "integer",
  "fk_elemento": "integer"
}
```
