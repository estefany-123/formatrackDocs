<h3>Encabezados de Solicitud</h3>

| Encabezado    | Valor          | Descripcion                                           |
| ------------- | -------------- | ----------------------------------------------------- |
| Authorization | Bearer `token` | Token con autorizacion para acceder a los inventarios |

<h3>Parametros del cuerpo</h3>

| Parametro y Tipo     | Descripcion                                                                                                                                              |
| -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| stock (integer)      | Cantidad de dicho elemento en bodega                                                                                                                     |
| estado (boolean)     | Si el elemento esta activo o inactivo                                                                                                                    |
| createdAt (date)     | Fecha en la que se registro el elemento en el inventario                                                                                                 |
| updatedAt (date)     | Fecha en la que se le hizo alguna modificacion en el inventario                                                                                          |
| fkSitio (integer)    | ID del sitio en el que se encuentra el inventario. Para ver los sitios existentes visite el siguiente endpoint <a href="/sitios/listarSitios">Sitios</a> |
| fkElemento (integer) | ID del elemento que seregistrara al inventario. Conozca los elementos disponibles aqui <a href="/elementos/listarElementos">elementos</a>                |

### Body

```json
{
  "stock": "integer",
  "estado": "string",
  "fkSitio": "integer",
  "fkElemento": "integer"
}
```
