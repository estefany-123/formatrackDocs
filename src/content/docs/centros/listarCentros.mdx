<h2>¿Que son los centros?</h2>

Los centros son unidades academicas y operativas donde se desarrollan los programas de formacion técnica, tecnológica y complementaria que brinda la institución.Cada uno de estos centros se especializan en diferentes sectores productivos y areas del conocimiento como la salud, la inductria, el agro entre otros.


<h3>Descripcion</h3>

Este endpoint permite obtener la lista de centros disponibles en el sistema.

Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/centros
  ```
  </TabItem>
</Tabs>


<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los centros
