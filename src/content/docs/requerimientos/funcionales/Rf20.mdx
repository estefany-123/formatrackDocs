<h3>Descripción</h3>
Para poder realizar el respectivo registro de un programa de formación deberá ingresar
en el botón (añadir Programa) de color azul ubicado en la parte superior izquierda
de la vista al ingresar se desplegará un formulario en donde deberá ingresar el nombre
del programa, el estado y el área luego dará click en guardar y así el programa se
creará correctamente.

<h3>Datos a ingresar para poder realizar el respectivo registro</h3>
Para poder registrar un Programa se necesita la siguiente información Nombre del
programa, estado, área
