<h3>Entradas</h3>
Se deberá ingresar el nuevo nombre del programa.

<h3>Descripción</h3>
Para poder actualizar un programa deberá dirigirse al icono de lápiz de color azul ubicado en la tabla, en el encabezado de acciones.  
Al dar clic se desplegará un modal con el input correspondiente donde se podrá modificar el nombre del programa.  
Posteriormente deberá dar clic en el botón "Guardar" para que se actualice correctamente.

<h3>Salidas</h3>
Mensaje de actualización exitosa y redireccionamiento al listado de programas.
