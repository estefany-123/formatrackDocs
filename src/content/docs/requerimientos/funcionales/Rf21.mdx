<h3>Descripción</h3>
Para poder actualizar un programa deberá dirigirse al icono de lapis de color azul
ubicado en la tabla en el encabezado de acciones de la misma.

<h3>Datos a ingresar para poder realizar el respectivo Actualización</h3>
Para poder realizar la actualización de un programa solo debera ingregar el nuevo
nombre del programa seleccionada
