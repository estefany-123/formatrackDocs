<h3>Entradas</h3>
El administrador podrá actualizar un usuario con: Nombre, Apellido, Edad, Teléfono,
Correo, Cargo

<h3>Descripción</h3>
El administrador podrá ver un icono de un lápiz con el cual podrá ver los datos del
usuario seleccionado y, en el modal que le aparece, estarán los inputs con la información
modificable. En la parte inferior estará el botón de guardar. Este requerimiento
hace parte del módulo de admin.

<h3>Salidas</h3>
Muestra la lista de todos los usuarios.

<h3>Prioridad</h3>
Alta
