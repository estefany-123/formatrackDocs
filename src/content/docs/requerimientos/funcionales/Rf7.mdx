<h3>Entradas</h3>
Se modificará el campo estado del usuario.

<h3>Descripción</h3>
El administrador podrá ver un icono de una basura con el cual podrá modificar el
estado del usuario (si está activo pasa a inactivo y viceversa). Este requerimiento
hace parte del módulo de admin.

<h3>Salidas</h3>
Muestra la lista de todos los usuarios.
