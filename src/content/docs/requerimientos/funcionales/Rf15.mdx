<h3>Entradas</h3>
Para poder registrar un área se necesita la siguiente información: Nombre, Estado,
Sede y Usuario.

<h3>Descripción</h3>
En este apartado principalmente se listarán las áreas ya registradas en el software.
Luego podrá realizar el respectivo registro ingresando al formulario, donde deberá
indicar los datos mencionados para completar el registro.

<h3>Salidas</h3>
Área registrada exitosamente y reflejada en la tabla de áreas.
