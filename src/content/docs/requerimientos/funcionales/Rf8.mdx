<h3>Entradas</h3>
Información del Sistema

<h3>Descripción</h3>
Esta es la vista general de todos los usuarios del sistema donde además tiene: un
botón en la parte superior izquierda que es para añadir un usuario, dentro de la
tabla se encontrará con dos botones de acciones: un ícono de lápiz (pencil) para
actualizar la información de un usuario y un ícono de basura (trash) para cambiar
el estado. Por otro lado, también tiene un botón en la parte superior derecha que
es para gestionar lo que tiene que ver con los roles disponibles que van relacionados
con la tabla de los usuarios. Además, en la parte superior de la tabla tiene una
opción para buscar algún usuario específico, otros dos para filtrar por las fechas
deseadas y también una opción de paginación para elegir la cantidad de registros
que se quieren ver. Este requerimiento hace parte del módulo de admin.

<h3>Salidas</h3>
Muestra la lista de todos los usuarios.
