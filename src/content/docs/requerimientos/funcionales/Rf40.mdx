<h3>Entradas</h3>
El administrador podrá registrar un tipo de sitio con: Nombre, estado

<h3>Descripción</h3>
En la parte superior derecha se encuentra un botón “Añadir Tipo de Sitio” con el
cual el administrador podrá ver un modal con los inputs para llenar con la información
de un nuevo tipo de sitio y, en la parte inferior, un botón de guardar. Este requerimiento
hace parte del módulo de admin.

<h3>Salidas</h3>
Muestra la lista de todos los tipos de sitios
