<h3>Entradas</h3>
El administrador podrá actualizar un tipo de sitio con: Nombre

<h3>Descripción</h3>
En la parte derecha de la tabla donde se encuentran los tipos de sitios, el administrador
podrá ver un botón de pencil el cual desplegará un modal con el input del tipo de
sitio seleccionado para que lo pueda editar, y en la parte inferior un botón de guardar.
Este requerimiento hace parte del módulo de admin.

<h3>Salidas</h3>
Muestra la lista de todos los tipos de sitios
