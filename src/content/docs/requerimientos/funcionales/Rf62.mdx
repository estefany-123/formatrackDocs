<h3>Entradas</h3>
Se modificará el campo estado del elemento en el inventario.

<h3>Descripción</h3>
Se realizará la modificación de los campos requeridos para la actualización, en este
caso el stock. Descripción En la vista de inventarios, en las acciones de la tabla
encontraremos el icono de una caja el cual hace referencia a el stock, entonces cuando
los elementos en el inventario no requieren de códigos para registrar el stock se
establecerá el formulario de actualización para este caso y se deberá definir la
cantidad de stock que posee el elemento.

<h3>Salidas</h3>
Mensaje de Estado cambiado con éxito.
