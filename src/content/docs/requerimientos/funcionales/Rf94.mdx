<h3>Descripción</h3>
Este apartado permite al administrador **actualizar el código de un inventario**
ya registrado. En la tabla de inventario, cada elemento tiene una opción de edición
para sus códigos individuales.

Al hacer clic en el botón azul con ícono de lápiz, se abrirá un pequeño formulario donde se puede editar directamente el código del inventario asociado al elemento seleccionado.

<h3>Entradas</h3>- Código actual del inventario (visible en el input) - Nuevo
código del inventario (a modificar)

<h3>Instrucciones para actualizar un código</h3>
1. Ingrese a la sección de **Inventario** en el módulo de Bodega. 2. Busque el elemento
cuyo código desea editar. 3. Haga clic en el botón azul con el ícono de editar (✏️).
4. Ingrese el nuevo código en el campo correspondiente. 5. Presione el botón **Guardar**
para confirmar los cambios.

<h3>Salidas</h3>- El código del inventario se actualiza correctamente. - El
nuevo valor se refleja inmediatamente en la tabla de inventario.
