<h3>Entradas</h3>
El administrador podrá registrar una categoría con: Nombre, estado

<h3>Descripción</h3>
En la parte superior derecha se encuentra un botón “Añadir Categoría” con el cual
el administrador podrá ver un modal con los inputs para llenar con la información
de una nueva categoría y, en la parte inferior, un botón de guardar. Este requerimiento
hace parte del módulo de bodega.

<h3>Salidas</h3>
Muestra la lista de todas las categorías
