<h3>Descripción</h3>
En la vista de elementos encontraremos tres inputs en la parte superior derecha de
la tabla en donde encontraremos el input de buscar para ingresar el texto correspondiente
a buscar y en los otros dos input estará para establecer un limite de fechas en la
cual vamos a filtrar la información. Esta son la fecha de inicio y la fecha fin.
Podremos filtrar la información que queramos en el rango de fechas que libremente
definamos y sea más fácil su búsqueda

<h3>Instrucciones para realizar la busqueda de un Elemento </h3>
Se deberá ingresar texto en el input de buscar para filtrar los elementos por nombre,
descripción, tipo de elemento. En caso tal también se podrá filtrar los elementos
que hay registrados en un rango de fechas.
