<h3>Entradas</h3>
Información del sistema.

<h3>Descripción</h3>
En esta vista se muestra una tabla con la información de las fichas, incluyendo:
- Código de ficha - Fecha de creación - Fecha de actualización - Estado - Acciones
(editar y cambiar estado)

La tabla cuenta con paginación para evitar saturar la vista si hay muchas fichas registradas.  
También permite filtrar por nombre (código de ficha) y por fechas (inicio y fin).

En la parte superior izquierda se encuentra un botón azul llamado **"Añadir ficha"**, el cual abre un formulario para registrar una nueva ficha.  
En la parte superior derecha, el botón **"Gestionar Programas"** redirige a la vista de gestión de programas de formación.

<h3>Salidas</h3>
Muestra la lista de todas las fichas registradas.
