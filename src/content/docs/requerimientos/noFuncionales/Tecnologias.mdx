## 🚀 Tecnologías Utilizadas en el Proyecto

El sistema **Formatrack** se encuentra dividido en dos partes fundamentales: **Frontend** y **Backend**, cada uno desarrollado con tecnologías modernas y ampliamente utilizadas en la industria del software.

---

### 🖥️ Frontend

Desarrollado con **React** y potenciado por **Vite**, el frontend ofrece una experiencia de usuario rápida y moderna. Las principales tecnologías y librerías utilizadas incluyen:

- **React 18** – Biblioteca principal para la construcción de interfaces de usuario.
- **Vite** – Herramienta de desarrollo rápido y eficiente.
- **Tailwind CSS** – Framework de utilidades para diseño responsivo y personalización visual.
- **HeroUI** – Conjunto de componentes visuales preconstruidos y accesibles.
- **React Hook Form** – Manejo de formularios eficiente y validación.
- **Zod** – Validación de esquemas de datos de forma tipada.
- **React Query** – Manejo de estado asincrónico y caché de datos.
- **Chart.js + React Chart.js 2** – Visualización de datos mediante gráficos interactivos.
- **Lucide React** – Librería de íconos SVG personalizables.
- **React PDF Renderer & Viewer** – Generación y visualización de documentos PDF.
- **Axios** – Cliente HTTP para consumir la API del backend.
- **Universal Cookie / JS-Cookie** – Gestión de cookies para sesiones de usuario.

---

### 🛠️ Backend

El backend está construido con el framework **NestJS**, ofreciendo una arquitectura escalable y mantenible basada en controladores, servicios e inyección de dependencias. A continuación, las tecnologías clave:

- **NestJS 11** – Framework de Node.js con arquitectura modular y soporte para TypeScript.
- **TypeORM** – ORM que facilita el acceso y manejo de la base de datos PostgreSQL.
- **PostgreSQL** – Base de datos relacional robusta y segura.
- **JWT (jsonwebtoken)** – Implementación de autenticación mediante tokens.
- **bcrypt** – Hashing de contraseñas seguro.
- **Nodemailer** – Envío de correos electrónicos (por ejemplo, para recuperar contraseñas).
- **Multer** – Manejo de archivos, como la carga de Excel.
- **xlsx** – Lectura y escritura de archivos Excel para registros masivos.
- **Socket.IO** – Comunicación en tiempo real mediante WebSockets.
- **Class-validator / class-transformer** – Validación y transformación de datos.
- **TypeScript** – Lenguaje de programación tipado para robustez y claridad de código.

---

### 🧪 Herramientas de Desarrollo

Además, se han utilizado múltiples herramientas para garantizar calidad de código y productividad:

- **ESLint / Prettier** – Linting y formateo automático del código.
- **Jest** – Framework de pruebas para verificar funcionalidades.
- **Vite TSConfig Paths** – Soporte para rutas absolutas en el frontend.
- **TS-Jest** – Integración de Jest con TypeScript.

---

Este conjunto de herramientas y tecnologías permite que el sistema Formatrack sea **escalable**, **modular** y **de fácil mantenimiento**, cumpliendo con los estándares modernos de desarrollo web.
