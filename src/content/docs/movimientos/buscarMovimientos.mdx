Este endpoint permite buscar movimientos específicos por su estado.

<p> 
<strong>Método: </strong><span class="sl-badge note small astro-avdet4wd">PUT</span>
</p>

<h3>Endpoint</h3>

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/api/movimiento/:estado
  ```
  </TabItem>
</Tabs>

Es necesario el envío del token de acceso <code>access_token</code> para hacer uso de este endpoint. Para ver cómo generar el token, vea <a href="/usuarios/login">como generar token de acceso.</a></aside>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los movimientos
