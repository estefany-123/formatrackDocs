<h2>Â¿Que son los movimientos?</h2>

Los movimientos de materiales de formacion se refiere a las acciones de traslado, ingreso o salida de los materiales o insumos que se usan en los procesos de formacion.

<h3>Descripcion</h3>

Este endpoint permite listar todos los movimientos registrados.


Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/api/movimiento/
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los movimientos
