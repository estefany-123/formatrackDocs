| -------- | ---------------------------------------------------------------------------- |
| URL      | [/api/movimiento/stock/:id_movimiento](/api/movimiento/stock/:id_movimiento) |
| Method   | `PUT`                                                                        |
| Headers  | Authorization : Bearer `token`                                               |
