<h3>Parametros del cuerpo</h3>

| Parametro y Tipo           | Descripcion                                                                                                                                                               |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| descripcion (string)       | Descripcion del movimiento y de lo que se hara.                                                                                                                           |
| cantidad (integer)         | cantidad requerida en el movimiento.                                                                                                                                      |
| horaIngreso (time)         | Si se realizo un tipo de movimento de ingreso se marca la hora en la cual se realizo el movimiento.                                                                       |
| horaSalida (time)          | Si se realizo un tipo de movimiento de salida se marca la hora en la cual se realizo el movimiento.                                                                       |
| aceptado (boolean)         | En caso de ser aceptado el movimiento.                                                                                                                                    |
| enProceso (boolean)        | Cuando el movimiento se encuentra en proceso.                                                                                                                             |
| cancelado (boolean)        | Se estblece cuando se ha rechazado el movimiento.                                                                                                                         |
| devolutivo (boolean)       | Se determina que el elemento solicitado del inventraio es de estado devolutivo                                                                                            |
| noDevolutivo (boolean)     | Se determina que el elemento solicitado del inventario es de estado no_devolutivo.                                                                                        |
| fechaDevolucion (date)     | Fecha de devolución de dicho elemento.                                                                                                                                    |
| lugarDestino (string)      | Lugar de destino del movimento a realizar.                                                                                                                                |
| createdAt (date)           | fecha en la que se solicito el movimiento.                                                                                                                                |
| updatedAt (date)           | fecha en lacual se actuallzo el movimiento.                                                                                                                               |
| fkSitio (integer)          | ID del sitio al que se va a hacer el movimiento. Para ver los sitios disponibles visite este endpoint <a href="/sitios/listarSitios">Sitios</a>                           |
| fkUsuario (integer)        | Id del usuario que solicito el movimiento. Para ver los usuarios visite este endpoint <a href="/usuarios/listarUsuarios">usuarios</a>                                     |
| fkTipoMovimiento (integer) | ID del tipo de movimiento que se hace. Para ver los tipos de movimientos visite este endpoint <a href="/tipos-movimiento/listarTipoMovimiento">tipos movimientos</a>      |
| fkInventario (integer)     | ID del inventario que se solicita para realizar el movimiento del elemento. Ver los elementos del inventario aqui <a href="/inventario/listarInventarios">Inventarios</a> |
