---
title: "Cambiar Estado de Tipo de Sitio"
description: "Permite activar o desactivar un tipo de sitio cambiando su estado."
slug: "tipos-sitio/cambiarEstadoTipoSitio"
---

Este endpoint permite cambiar el estado de un tipo de sitio (activo/inactivo) utilizando su ID.

| Variable | Valor                                          |
| -------- | ---------------------------------------------- |
| URL      | [/tipoSitio/estado/:id](/tipoSitio/estado/:id) |
| Method   | `PUT`                                          |
| Headers  | Authorization : Bearer `token`                 |
