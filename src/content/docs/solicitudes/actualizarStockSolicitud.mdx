| -------- | ------------------------------------------------------------------------ |
| URL      | [/api/solicitud/stock/:id_solicitud](/api/solicitud/stock/:id_solicitud) |
| Method   | `PUT`                                                                    |
| Headers  | Authorization : Bearer `token`                                           |
