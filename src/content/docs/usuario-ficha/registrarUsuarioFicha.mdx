<h3>Encabezados de Solicitud</h3>

| Encabezado    | Valor            | Descripcion                                          |
| ------------- | ---------------- | ---------------------------------------------------- |
| Content-Type  | application/json | Indica que los datos esran manejados en fromato JSON |
| Authorization | Bearer `token`   | Token con autorizacion para acceder a las areas      |

<h3>Endpoint para los Datos de usuario ficha</h3>

Para agilizar la creacion de nuevos usurios ficha te ofrecemos estos endpoints de apoyo para que puedas consultar y obtener el ID unico tando de fichas como de usuarios, los cuales son necesarias para crear tu area.

  <a href="/fichas/listarFichas">Fichas</a>

<li>

<h3>Parametros del cuerpo</h3>

| Parametro y Tipo    | Descripcion                                                                                                                                                 |
| ------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| fkFicha (number)    | ID de la ficha a la cual pertence el usuario. Para consultar las fichas existentes consulte este endpoint <a href="/Fichas/listarFichas">fichas</a>         |
| fk_usuario (number) | ID del usuario al usuario ficha perteneciente. Para consultar los usuario existentes consulte este endpoint <a href="/usuarios/listarUsuarios">usuarios</a> |
| createdAt (date)    | fecha de creació del usuari ficha.Esta se crea automaticamente                                                                                              |
| updatedAt (date)    | fecha de actualización del ar usuari fichaa.Esta se crea automaticamente                                                                                    |

### Body

```json
{
  "fkFicha": "integer",
  "fkUsuario": "integer"
}
```
