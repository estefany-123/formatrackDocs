<h3>Encabezados de Solicitud</h3>

| Encabezado    | Valor          | Descripcion                                       |
| ------------- | -------------- | ------------------------------------------------- |
| Authorization | Bearer `token` | Token con autorizacion para acceder a los centros |

<h3>Endpoint para los Datos del Usuario</h3>

Para agilizar la creacion de nuevos usuarios te ofrecemos este endpoint de apoyo para que puedas consultar y obtener el ID del rol necesaria para crear tu usuario.

<li>
  <a href="/roles/listarRoles">roles</a>
</li>

<h3>Parametros del cuerpo</h3>

| Parametro y Tipo   | Descripcion                                                                                                                                                |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| documento (string) | documento del usuario a crear                                                                                                                              |
| nombre (string)    | Nombre del usuario                                                                                                                                         |
| apellido (string)  | apellido del usuario                                                                                                                                       |
| edad (number)      | edad del usuario                                                                                                                                           |
| estado (string)    | Si se encuentra activo o inactiva                                                                                                                          |
| cargo (boolean)    | cargo al que se le asigna al usuario                                                                                                                       |
| password (string)  | password del usuario                                                                                                                                       |
| perfil (string)    | perfil del usuario                                                                                                                                         |
| created_at (date)  | Fecha en la que se registro el usuario                                                                                                                     |
| updated_at (date)  | Ultima fecha en la que se le hizo alguna modificacion al usuario                                                                                           |
| fkrol (number)     | ID del rol al cual pertenece el usuario. Para saber cual es el id del rol registrado consulte el siguiente endpoint <a href="/roles/listarRoles">roles</a> |

### Body

```json
{
  "documento": "string",
  "nombre": "string",
  "apellido": "string",
  "edad": "number",
  "telefono": "string",
  "correo": "string",
  "estado": "boolean",
  "cargo": "string",
  "password": "string",
  "perfil": "string",
  "fkrol": "number"
}
```
