Este endpoint permite registrar múltiples usuarios de forma masiva mediante un archivo Excel cargado por el administrador.

<p>
  <strong>Método: </strong>
  <span class="sl-badge success small astro-avdet4wd">POST</span>
</p>

<Tabs>
  <TabItem label="Endpoint">
    <ApiEndpoint url={`${API_BASE_URL}/usuarios/massive`} />
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado    | Valor               | Descripción                                                  |
| ------------- | ------------------- | ------------------------------------------------------------ |
| Content-Type  | multipart/form-data | Indica que el archivo Excel se está cargando como formulario |
| Authorization | Bearer `token`      | Token con autorización para acceder al recurso               |
