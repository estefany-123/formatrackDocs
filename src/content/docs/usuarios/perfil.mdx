Este endpoint permite obtener la información del **perfil del usuario autenticado**, incluyendo sus datos personales y el rol asociado. Es necesario enviar un token de acceso válido para su correcta ejecución.

<p>
  <strong>Método: </strong>
  <span class="sl-badge success small astro-avdet4wd">GET</span>
</p>

<Tabs>
  <TabItem label="Endpoint">
    <ApiEndpoint url={`${API_BASE_URL}/usuarios/perfil`} />
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado    | Valor          | Descripcion                                   |
| ------------- | -------------- | --------------------------------------------- |
| Authorization | Bearer `token` | Token con autorización para acceder al perfil |
