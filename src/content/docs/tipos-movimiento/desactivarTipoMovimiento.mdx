---
title: "Desactivar Tipo de Movimiento"
description: "Permite activar o desactivar un tipo de movimiento cambiando su estado."
slug: "tipos-movimiento/desactivarTipoMovimiento"
---

Este endpoint permite cambiar el estado de un tipo de movimiento (activo/inactivo) utilizando su ID.

| Variable | Valor                                                                              |
| -------- | ---------------------------------------------------------------------------------- |
| URL      | [/api/tipoMovimiento/desactivar/:id_tipo](/api/tipoMovimiento/desactivar/:id_tipo) |
| Method   | `PUT`                                                                              |
| Headers  | Authorization : Bearer `token`                                                     |
