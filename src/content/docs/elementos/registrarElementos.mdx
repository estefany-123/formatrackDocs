<strong>Método: </strong><span class="sl-badge note small astro-avdet4wd">PUT</span>
</p>

<Tabs>

  ```
  http://localhost:3000/api/elemento/register/
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los elementos

<h3>Endpoint para los Datos del Elemento</h3>

<h3>Parametros del cuerpo</h3>


| Parametro y Tipo | Descripcion
|----------|--------
| nombre (string) |  Nombre por defecto del elemento
| descripcion (string) | Algunos detalles adicionales del elemento
| valor (integer) | valor por la unidad del elemento
| consumible (boolean) | Determina que el elemento es consumible
| no_consumible (boolean) | Determina que el elemento es no consumible
| estado (boolean) | Si se encuentra activo o inactivo el elemento
| imagen_elemento (string) | Imagen relacionada con el elemento a registrar
| fecha_creacion (date) | Fecha en la que se registro el elemento
| fecha_actualizacion (date) | Ultima fecha en la que se le hizo alguna modificacion
| fk_unidad_medida (integer) | ID de la unidad de medida con la que se identifica dicho elemento. Para conocer las unidades de medida existentes puede ir al este endpoint <a href="/unidades-medida/listarUnidadesMedida">unidad medida</a>
| fk_categoria (integer) | ID de la categoria a la que pertenece el elemento. Para conocer las categorias puede consultar en este endpoint <a href="/categorias/listarCategorias">categorias</a>
| fk_caracteristica (integer) | ID de la caracteritica a la que pertenece el elemento. Para conocer las caracteristicas puede consultar en este endpoint <a href="/caracteristicas/listarCaracteristicas">caracteristica</a>

### Body

```json
{
  "nombre": "string",
  "descripcion": "string",
  "valor": "integer",
  "consumible": "boolean",
  "no_consumible": "boolean",
  "estado": "boolean",
  "imagen_elemento": "string",
  "created_at": "date",
  "updated_at": "date",
  "fk_unidad_medida": "integer",
  "fk_categoria": "integer"
  "fk_caracteristica": "integer"
}
```
