<h3>Encabezados de Solicitud</h3>

| Encabezado    | Valor          | Descripcion                                         |
| ------------- | -------------- | --------------------------------------------------- |
| Authorization | Bearer `token` | Token con autorizacion para acceder a los elementos |

<h3>Endpoint para los Datos del Elemento</h3>

<h3>Parametros del cuerpo</h3>

| Parametro y Tipo           | Descripcion                                                                                                                                                                                                   |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| nombre (string)            | Nombre por defecto del elemento                                                                                                                                                                               |
| descripcion (string)       | Algunos detalles adicionales del elemento                                                                                                                                                                     |
| perecedero (boolean)       | Determina que el elemento es perecedero                                                                                                                                                                       |
| noPerecedero (boolean)     | Determina que el elemento es no perecedero                                                                                                                                                                    |
| estado (boolean)           | Si se encuentra activo o inactivo el elemento                                                                                                                                                                 |
| imagenElemento (string)    | Imagen relacionada con el elemento a registrar                                                                                                                                                                |
| fechaUso (date)            | Fecha de uso del elemento                                                                                                                                                                                     |
| fkCategoria (date)         | ID de la categoria a la cual se encuentra relacionada el elemento. Para conocer las categorias existentes puede dirigirse a este endpoint <a href="/categorias/listarCategorias">Categorias</a>               |
| fkUnidadMedida (integer)   | ID de la unidad de medida con la que se identifica dicho elemento. Para conocer las unidades de medida existentes puede ir al este endpoint <a href="/unidades-medida/listarUnidadesMedida">unidad medida</a> |
| fkCaracteristica (integer) | ID de la caracteristica a la que pertenece el elemento. Para conocer las caracteristicas puede consultar en este endpoint <a href="/caracteristicas/listarCaracteristicas">caracteristicas</a>                |
| createdAt (date)           | fecha de creacion del elemento. Esta se genera automaticamente.                                                                                                                                               |
| updatedAt (date)           | fecha de actualización del elemento. Esta se genera automaticamente                                                                                                                                           |

### Body

```json
{
  "nombre": "string",
  "descripcion": "string",
  "perecedero": "boolean",
  "noPerecedero": "boolean",
  "estado": "boolean",
  "imagenElemento": "string",
  "fechaUso": "date",
  "fkCategoria": "number",
  "fkUnidadMedida": "number",
  "fkCaracteristica": "number"
}
```
