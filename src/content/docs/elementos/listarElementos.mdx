<h2>Â¿Que son los elementos?</h2>
Los elementos son los componentes escenciales que ayudan a que los aprendices tengan los recursos materiales necesarios para que adquieran sus habilidades necesarias

<h3>Descripcion</h3>

Este endpoint permite obtener la lista de elementos disponibles en el sistema.

Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/api/elemento/
  ```
  </TabItem>
</Tabs>


<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los elementos
