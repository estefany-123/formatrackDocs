<h2>¿Que son los elementos?</h2>

Los Elementos son las unidades físicas o materiales registradas en el sistema que
pueden ser asignadas, prestadas, almacenadas o gestionadas dentro del inventario
de la institución.

<h3>Descripcion</h3>

Este endpoint permite obtener la lista de elementos disponibles en el sistema.

<p>
  <strong>Método: </strong>
  <span class="sl-badge success small astro-avdet4wd">GET</span>
</p>

<Tabs>
  <TabItem label="Endpoint">
    <ApiEndpoint url={`${API_BASE_URL}/elementos`} />
  </TabItem>
</Tabs>
