import { Tabs, TabItem } from '@astrojs/starlight/components';

Este endpoint permite buscar un programa de formación específico por su nombre.

Methodo:`GET`

<h3>Endpoint</h3>

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/P.formacion/:nombre
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los programas de formacion

<h3>Ejemplo de respuesta</h3>

<Tabs>
  <TabItem label="Status 200">
  ```json
{
    "id_programa": 3,
    "nombre": "multimedia",
    "estado": false,
    "created_at": "3038-10-10T05:00:00.000Z",
    "updated_at": "3034-10-10T05:00:00.000Z",
    "fk_area": 3
  }
```
  </TabItem>

  <TabItem label="Status 404">
  ```json
  {message:"programa no existe"}
  ```
  </TabItem>
</Tabs>
