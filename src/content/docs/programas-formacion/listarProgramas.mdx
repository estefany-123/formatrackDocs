<h2>¿Que son las Areas?</h2>

Los programas de formación son planes de estudio diseñados para capacitar a los aprendices en diferentes áreas del conocimiento y prepararlos para el mundo laboral

<h3>Descripcion</h3>

Este endpoint permite listar todos los programas de formación registrados.


Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/P.formacion
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los programas de formacion
