import { Tabs, TabItem } from '@astrojs/starlight/components';


Este endpoint permite actualizar la información de una categoría específica mediante su ID.

Method: `PUT`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/categorias/:id
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a las categorias


### Body

```json
{
  "nombre": "string",
  "estado" : "boolean",
  "created_at": "date",
  "updated_at": "date"
}
```

<h3>Ejemplos de respuesta</h3>

<Tabs>
  <TabItem label="Status 200">
  ```json
  {message:"Actualizado con exito"}
  ```
  </TabItem>

  <TabItem label="Status 500">
  ```json
  {message:"Error actualizando"}
  ```
  </TabItem>
</Tabs>
