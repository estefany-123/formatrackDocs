<h2>¿Que son las categorias?</h2>

Las categorias es la forma en la que clasificamos los elementos como por ejemplo pesticidas, alimentos, equipos electronicos y demas.

<h3>Descripcion</h3>

Este endpoint permite obtener la lista de categorías disponibles en el sistema.

Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/categorias
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a las categorias
