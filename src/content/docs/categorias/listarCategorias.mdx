<h2>¿Que son las categorias?</h2>

Las categorías representan agrupaciones generales de elementos o materiales que comparten características comunes. Estas ayudan a organizar el inventario de manera estructurada, facilitando su identificación y gestión. Por ejemplo, una categoría puede ser "Herramientas Eléctricas", "Material Didáctico" o "Equipos de Protección Personal".
Además, cada categoría está relacionada con un código UNPSC, que es un estándar internacional para clasificar productos y servicios, lo que permite una mejor trazabilidad y uniformidad en los registros.

<h3>Descripcion</h3>

Este endpoint permite obtener la lista de categorías disponibles en el sistema.
