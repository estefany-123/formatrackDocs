<h3>Parametros del cuerpo</h3>

| Parametro y Tipo | Descripcion |
| ---------------- | ----------- |

| fkPermiso (number) ID del permiso al cual pertenece el rol permiso proximo a crear|
| fkRol (number) ID del rol al cual pertenece el rol permiso |
| estado (boolean) | Si se encuentra activo o inactiva el area |
| createdAt (date) | fecha de creació del area.Esta se crea automaticamente |
| updatedAt (date) | fecha de actualización del area.Esta se crea automaticamente |

### Body

```json
{
  "fkPermiso": number,
  "fkRol": number,
  "estado": boolean
}
```
