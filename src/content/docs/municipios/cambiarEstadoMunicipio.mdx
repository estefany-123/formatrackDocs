Este endpoint permite cambiar el estado de un municipio (activo/inactivo) utilizando su ID.


Method : `PUT`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/municipios/estado/:id
  ```
  </TabItem>
</Tabs>


<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los municipios



<h3>Ejemplos de respuesta</h3>

<Tabs>
  <TabItem label="Status 200">
  ```json
  {message:"Estado actualizado con exito"}
  ```
  </TabItem>
</Tabs>
