<h2>¿Que son los municipios?</h2>

Un municipio es una entidad territorial y administrativa que puede agrupar una o varias localidades. Es la unidad básica de organización territorial del Estado

<h3>Descripcion</h3>

Este endpoint permite listar todos los municipios registrados.



Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/municipios
  ```
  </TabItem>
</Tabs>

<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a los municipios
