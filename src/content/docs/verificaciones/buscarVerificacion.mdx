---
title: "Buscar Verificación por Fecha"
description: "Busca verificaciones en el sistema utilizando la fecha de verificación."
slug: "verificaciones/buscarVerificacion"
---

Este endpoint permite buscar verificaciones específicas por su fecha de verificación.

| Variable | Valor                                                                          |
| -------- | ------------------------------------------------------------------------------ |
| URL      | [/api/verificacion/:fecha_verificacion](/api/verificacion/:fecha_verificacion) |
| Method   | `GET`                                                                          |
| Headers  | Authorization : Bearer `token`                                                 |
