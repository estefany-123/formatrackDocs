<h2>¿Que son las Caracteristicas?</h2>

El módulo de Características permite registrar y administrar atributos específicos que definen o describen a los elementos materiales del sistema. Estas características permiten una clasificación más detallada de los elementos, facilitando su identificación, organización y búsqueda dentro del inventario.

Cada característica puede estar asociada a múltiples elementos y puede incluir propiedades como dimensiones, color, modelo, marca, voltaje, material, entre otros, dependiendo del tipo de recurso o material gestionado.

<h3>Descripcion</h3>

Este endpoint permite obtener la lista de las Caracteristicas disponibles en el sistema.
