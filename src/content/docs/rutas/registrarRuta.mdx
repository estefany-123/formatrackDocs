<h3>Encabezados de Solicitud</h3>

| Encabezado    | Valor            | Descripcion                                          |
| ------------- | ---------------- | ---------------------------------------------------- |
| Content-Type  | application/json | Indica que los datos esran manejados en fromato JSON |
| Authorization | Bearer `token`   | Token con autorizacion para acceder a las areas      |

<h3>Endpoint para los Datos de las rutas</h3>

Para agilizar la creacion de nuevas rutas te ofrecemos estos endpoints de apoyo para que puedas consultar y obtener el ID unico del modulo el cual es necesario para crear tu ruta.

<li>
  <a href="/modulos/listarModulos">Modulo</a>
</li>

<h3>Parametros del cuerpo</h3>

| Parametro y Tipo     | Descripcion                                                                                                                                     |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| nombre (string)      | Nombre que se le va a designar a dicha ruta no mayor a (50)                                                                                     |
| description (string) | Descripción detallada de la ruta a crear                                                                                                        |
| estado (boolean)     | Si se encuentra activo o inactiva la ruta                                                                                                       |
| descripcion (string) | Descripcion respecto a la ruta a crear                                                                                                          |
| createdAt (date)     | fecha de creació de la ruta .Esta se crea automaticamente                                                                                       |
| updatedAt (date)     | fecha de actualización de la ruta.Esta se crea automaticamente                                                                                  |
| fkModulo (number)    | ID del modulo a la ruta perteneciente. Para consultar los modulos existentes consulte este endpoint <a href="/modulos/listarModulos">Modulo</a> |

### Body

```json
{
  "nombre": "string",
  "descripcion": "string",
  "urlDestino": "string",
  "estado": "boolean",
  "fkModulo": "number"
}
```
