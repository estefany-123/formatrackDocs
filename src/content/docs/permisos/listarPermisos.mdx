<h2>¿Que son los permisos?</h2>

La tabla permisos define los niveles de acceso que pueden tener los usuarios dentro del sistema. Cada registro en esta tabla representa una acción específica o conjunto de acciones permitidas, como visualizar, registrar, actualizar o eliminar información. Estos permisos son esenciales para establecer un control preciso sobre lo que cada usuario puede o no puede hacer en el sistema, asegurando así la integridad, seguridad y adecuada gestión de la información administrativa y operativa.

<h3>Descripcion</h3>

Este endpoint permite obtener la lista de los permisos disponibles en el sistema.

</p>
