<h2>Â¿Que son las fichas?</h2>

Las fichas de formacion son codigos o identificadores que se les asigna a cada grupo de aprendices pertenecientes a un programa de formacion. Estas permiten organizar, gestionar y hacer el siguimiento a los procesos y avances academicos.

<h3>Descripcion</h3>

Este endpoint permite obtener la lista de fichas disponibles en el sistema.


Method : `GET`

<Tabs>
  <TabItem label="Endpoint">
  ```
  http://localhost:3000/Fichas
  ```
  </TabItem>
</Tabs>


<h3>Encabezados de Solicitud</h3>

| Encabezado | Valor | Descripcion
|----------|-------| -------
| Authorization |  Bearer `token` | Token con autorizacion para acceder a las fichas
