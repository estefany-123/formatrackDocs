<h3>Encabezados de Solicitud</h3>

| Encabezado    | Valor            | Descripcion                                          |
| ------------- | ---------------- | ---------------------------------------------------- |
| Content-Type  | application/json | Indica que los datos esran manejados en fromato JSON |
| Authorization | Bearer `token`   | Token con autorizacion para acceder a las areas      |

<h3>Parametros del cuerpo</h3>

| Parametro y Tipo     | Descripcion                                                    |
| -------------------- | -------------------------------------------------------------- |
| nombre (string)      | Nombre que se le va a designar a dicho modulo no mayor a (50)  |
| estado (boolean)     | Si se encuentra activo o inactiva el modulo                    |
| descripcion (string) | Descripcion respecto al modulo a crear                         |
| createdAt (date)     | fecha de creació del modulo.Esta se crea automaticamente       |
| updatedAt (date)     | fecha de actualización del modulo.Esta se crea automaticamente |

### Body

```json
{
  "nombre": "string",
  "estado": "boolean",
  "descripcion": "string"
}
```
